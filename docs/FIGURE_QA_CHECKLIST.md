# Figure QA Checklist

Use this checklist before freezing any PR2 figure.

1. Does the figure title match the report claim?
2. Does the figure use only approved shared colors?
3. Do labels include units where needed?
4. Does the filename match `reporting.py`?
5. Do notebook values match the exported CSV table?
6. Does the dashboard show the same metric value?
7. If the figure is a map, are CRS and basemap correct?
8. If the figure is a change chart, does green mean improvement and red mean worsening?
9. Is the legend order consistent with other figures of the same type?
10. Is the figure readable at report scale?
11. If the figure uses employment data, is it labeled as jobs proxy rather than exact jobs?
